# BFS and DFS Graph Traversal Practical

## Aim

To implement Breadth First Search (BFS) and Depth First Search (DFS) traversal on a graph.

---

# What is Graph?

A graph is a data structure made of:

* Vertices (nodes)
* Edges (connections)

Example:

```text id="f0l4dn"
0 ---- 1
|      |
2      3
```

---

# What is DFS?

DFS means:

```text id="3drh7k"
Depth First Search
```

* DFS visits nodes deeply first.
* Uses recursion or stack.

Example:

```text id="ekntb7"
0 → 1 → 3 → backtrack → 4 → backtrack → 2 → 5
```

---

# What is BFS?

BFS means:

```text id="jqkm1k"
Breadth First Search
```

* BFS visits nodes level by level.
* Uses Queue data structure.

Example:

```text id="lf5ks2"
0 → 1 → 2 → 3 → 4 → 5
```

---

## Difference between BFS and DFS?

| BFS                  | DFS                              |
| -------------------- | -------------------------------- |
| Level wise traversal | Deep traversal                   |
| Uses Queue           | Uses Stack/Recursion             |
| Finds shortest path  | Does not guarantee shortest path |
